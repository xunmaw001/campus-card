# campus-card
基于Spring Boot和Vue.js的校园一卡通系统，旨在为高校师生提供便捷的校园服务。系统支持充值、消费、余额查询和交易记录管理，涵盖图书馆借阅、食堂消费和校园交通等功能。后端使用Spring Boot确保数据安全与稳定性，前端采用Vue.js构建直观易用的界面，提升用户体验。该系统简化了校园生活管理，增强了师生的便捷性与安全感，为智能校园建设提供有力支持。
